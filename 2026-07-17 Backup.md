### How to Use Library
```python
from qscout import QScout
import time

robot = QScout()
robot.connect()
print(f"Connected: {robot.is_connected}")

# Sensors
ultrasonic = robot.sensors.get_ultrasonic_distance()
print(f"Distance: {ultrasonic}mm")

# LED
robot.actuators.set_led(0, 255, 0)  # Green
time.sleep(1)
robot.actuators.set_led_off()

robot.disconnect()
```

## Key Technical Findings

### Protocol RB
- **Header:** `0x52 0x42` ("RB")
- **Checksum:** `sum(all_bytes) % 256`
- **Order ID:** Correlation mechanism for request-response matching
- **Action codes:** Request ≠ Response (0x03=device info, 0x01=ACK/generic)

### Hardware
- **Connection:** USB Serial via CH340 (VID:PID 1A86:7523)
- **Speed:** 115200 baud, 8N1
- **Firmware:** Arduino/ESP-IDF C++ (ESP-IDF v3.3.5 + Arduino Core 1.0.6)
- **Motor speed range:** -100..100 (clamped)

### Auto-Detection
```python
# Primary: VID:PID
CH340_VID = 0x1A86
CH340_PID = 0x7523

# Fallback: description
'ch340' in desc or 'usb-serial' in desc or 'usb serial' in desc
```
