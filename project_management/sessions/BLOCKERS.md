## How to Report a Blocker

1. Add entry to appropriate section above
2. Update TASK_STATE.yaml: `blocked: true`
3. Update TASK_STATE.yaml: `blocking_reason: [description]`
4. Notify Coordinator immediately

---

## How to Resolve a Blocker

1. Investigate root cause
2. Determine if resolvable by agents
3. If yes: implement fix
4. If no: escalate to human
5. Update this file when resolved
6. Update TASK_STATE.yaml: `blocked: false`
